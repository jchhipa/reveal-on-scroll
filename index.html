Scroll to reveal elements

How to Use

Give an element css property opacity 0 (class .reveal by default). 

.reveal { opacity: 0;}

Link and activate revealOnScroll.js 

<script src="js/reveal-on-scroll.min.js"></script>
new repeatOnScroll().init();

Options

selector Default: 'reveal' String A selector which identifies the element that should reveal on scroll.
delay Default: 100 Number Delay the fade animation
offset Default: 0 Number distance to the element when triggering the animation
animationSpeed Default: 1500 Number Set the speed of the fade animation

Example

var reveal = new revealOnScroll({
	selector: 'element',
    	delay: 500,
    	offset: 100,
    	animationSpeed: 200
}); 
reveal.init();
